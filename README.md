
Project Overview

This project was created independently to better understand how attendance behavior may differ across NFL clubs using publicly available datasets. The goal was to practice analytics workflows, feature engineering, exploratory modeling, segmentation, and dashboard development while exploring whether different attendance drivers emerge across markets.

Disclaimer

This is an independent educational project and is not affiliated with, endorsed by, or representative of the NFL. All analyses were conducted using publicly available data and should be considered exploratory in nature.



Business Questions:

Which clubs outperform league-average attendance?

What factors appear most associated with attendance?

Do attendance drivers differ by club?

Can clubs be segmented based on attendance behavior patterns?



Methodology:


Data collection from public datasets

Data cleaning and standardization

Data integration and joins

Feature engineering

League benchmarking

Driver analysis

Club segmentation

Dashboard development



Key Outputs:


League attendance benchmarking

Club-level attendance driver analysis

Attendance behavior segmentation

Interactive team profiles

Power BI dashboard



Future Enhancements:


Stadium capacity normalization

Why I Built This:

I built this project to strengthen my understanding of fan analytics, audience segmentation, and data storytelling. Rather than focusing solely on predictive modeling, the project emphasizes translating data into interpretable club-level insights and strategic recommendations.
